# RecoFlix 🎬

RecoFlix is a real-time movie recommendation web application built using Streamlit and the TMDB API.

## Features
- Browse popular movies
- View trending movies (weekly)
- Discover movies by genre
- Search for a movie and get similar recommendations
- Real-time data with graceful API error handling

## Tech Stack
- Python
- Streamlit
- TMDB API

## Live Demo
👉 https://recoflix.streamlit.app/

## Notes
RecoFlix uses the free TMDB API. Occasional missing posters or empty results may occur due to API limitations.
